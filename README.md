# luzuu-love
My loved one❣️# 💖 Love Story Web App – Birthday Surprise

## 📌 Description
This project is a creative **birthday surprise web application** designed to express emotions using web technologies.  
It presents a beautiful **auto-playing love story** with messages, images, animations, and music.

---

## 🎯 Features
- ✨ Typing text animation  
- 🌌 Starry (galaxy) background  
- 💬 Chat-style love story  
- 📸 Images displayed inside chat  
- 🎶 Background music  
- ❤️ Final birthday message  
- 🎬 Fully automatic (no user interaction needed except music)

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
